2. Write a Java program to create a method that takes an integer as a parameter and throws an exception if the number is odd.

class Odd
{
    	public static void checkEvenNumber(int number) throws OddNumberException 
	{
       		 if (number % 2 != 0) 
		{
           	 	throw new OddNumberException("The number " + number + " is odd.");
        	}
        	System.out.println("The number " + number + " is even.");
    	}

    public static class OddNumberException extends Exception {
        public OddNumberException(String message) {
            super(message);
        }
    }

    public static void main(String[] args) {
        try {
            checkEvenNumber(10); // This will print "The number 10 is even."
            checkEvenNumber(11); // This will throw an OddNumberException
        } catch (OddNumberException e) {
            System.out.println("Caught exception: " + e.getMessage());
        }
    }
}

Output

The number 10 is even.
Caught exception: The number 11 is odd.
