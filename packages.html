{
	public void printMessage()

	}

	}

class D

7.Create a Java program to create a nested package i.e. c4.sample with public class and protected members to be accessed in another class.

package c4.sample;

public class nine
{
	protected String protectedMessage="Hello from protected member";
	public String publicMessage="Hello from public member";

	protected void protectedMethod()
	{
		System.out.println("Hello its protected");
	}
	public void publicMethod()
	{
		System.out.println("Hello its public");
	}

}
class B extends nine
{
	public static void main(String args[])
	{
		B obj = new B();
		obj.protectedMethod();
		System.out.println(obj.protectedMessage);

		obj.publicMethod();
		System.out.println(obj.publicMessage);
		
	}
}

Output

D:\javaprogram\package>javac -d . nine.java

D:\javaprogram\package>java c4.sample.B
Hello its protected
Hello from protected member
Hello its public
Hello from public member
